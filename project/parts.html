PC Components

Picking the right parts can seem daunting, but it doesn't have to be!

Component Description Image
CPU
(Central Processing Unit)		 The brain of the computer. Performs most of the processing and runs instructions.
GPU
(Graphics Processing Unit)		 Handles graphics, video, and 3D images. Necessity for gaming, video editing, and AI projects.
RAM
(Random Access Memory)		 Short-term memory that stores data for active applications.
PSU
(Power Supply Unit)		 Converts electricity from the wall into usable power for internal components.
Motherboard The main circuit board that connects all the parts. Houses CPU, RAM, GPU, and storage.
Storage
(HDD/SSD)		 Holds files, software, and operating system.
Cooling System
(Fans/Heatsinks)		 Keeps temperatures within safe limits by cooling down components.
Case Contains and protects all components. Comes in various sizes!
